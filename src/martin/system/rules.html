Rules

Fundamental constraints that guide decision making.

1. Maximum Leverage

Always choose the path that yields the highest output per unit of input.

2. Defaults Matter

If it requires willpower, it will fail. Build the default path to be the right path.

3. Documentation is Product

If it isn't written down, it doesn't exist. Treat documentation with the same care as shipping code.
